Proyecto 2: Aplicación de Gestión

Esta aplicación web permite gestionar inventarios y ventas en tiempo real. Incluye funcionalidades como la administración de usuarios, generación de reportes y sincronización con múltiples dispositivos.

Se desarrolló utilizando el framework Laravel para el backend, lo que permitió una estructura sólida y escalable. La interfaz de usuario se diseñó con Bootstrap, logrando una experiencia de usuario fluida en cualquier dispositivo. El sistema de gestión de inventarios está vinculado a una base de datos relacional en MySQL, lo que facilita la búsqueda y actualización de productos de forma eficiente.

La aplicación también incorpora una funcionalidad avanzada de reportes, que permite generar informes personalizados sobre ventas, inventario y usuarios. Gracias a la integración con API REST, el sistema puede conectarse con aplicaciones externas, lo que le permite una mayor flexibilidad y adaptabilidad a diferentes necesidades del cliente.
